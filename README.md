# Python Scripts

A collection of small Python utility scripts built to practice programming fundamentals and problem-solving.

---

## Scripts

### 🔐 password_generator.py
Generates a secure random password based on user-defined length and character preferences (uppercase, lowercase, numbers, symbols).

**How to run:**
```
python password_generator.py
```

**Example output:**
```
Generated password: Xk#9mLqZ2!
```

---

## Requirements
- Python 3.x
- No external libraries needed (uses built-in `random` and `string` modules)
